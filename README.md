1. Introduction

Data visualization is an essential step in data analysis that converts raw data into meaningful graphical representations. After collecting data through web scraping, visualizations help identify patterns, trends, and insights more effectively. In this project, the scraped dataset was analyzed and visualized using Python libraries such as Matplotlib and Seaborn to improve understanding of the collected information.

2. Objective

The main objectives of this task are:

Transform scraped data into visual formats.
Identify trends and patterns in the dataset.
Improve data interpretation through graphical analysis.
Support decision-making using visual insights.
Create professional charts and graphs for reporting purposes.
3. Dataset Description

The dataset was collected from a publicly available website through web scraping techniques. The extracted data includes:

Product Name
Price
Rating
Availability
Category
Description
Date of Extraction
Website URL

The cleaned dataset was stored in CSV format and used for visualization.

6. Data Preparation for Visualization

Before creating visualizations, the following preprocessing steps were performed:

Imported the scraped dataset into Pandas.
Checked for missing values.
Removed duplicate records.
Converted data types where necessary.
Cleaned and formatted numerical values.
Organized data into structured tables.
6. Visual Analysis
6.1 Category Distribution Analysis

A bar chart was created to show the number of products available in each category.

Observation:

Some categories contained significantly more products than others.
The distribution helped identify dominant categories.
6.2 Price Distribution Analysis

A histogram was used to analyze product price distribution.

Observation:

Most products were concentrated within a specific price range.
A few products had exceptionally high prices.
6.3 Rating Analysis

A rating distribution chart was created to examine customer ratings.

Observation:

The majority of products received positive ratings.
Highly rated products were easily identified.
6.4 Availability Analysis

A count plot was generated to compare available and unavailable products.

Observation:

Most products were available at the time of data extraction.
Inventory trends became easier to understand.
6.5 Correlation Analysis

A heatmap was created for numerical variables.

Observation:

Relationships between numerical attributes were identified.
Correlation values highlighted possible dependencies.
7. Data Visualizations Created

The following visualizations were generated:

1. Category Distribution Bar Chart

Displays the number of products in each category.

2. Price Distribution Histogram

Shows the frequency distribution of product prices.

3. Rating Distribution Chart

Illustrates customer rating patterns.

4. Product Availability Count Plot

Shows available versus unavailable products.

5. Top Categories Bar Chart

Highlights categories with the highest number of products.

6. Correlation Heatmap

Displays relationships among numerical features.
